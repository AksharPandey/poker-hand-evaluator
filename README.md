<img width="617" height="464" alt="image" src="https://github.com/user-attachments/assets/83eef8e6-bccf-4ddb-8921-44beb93c7242" /># Poker Hand Evaluator
Learning Monte Carlo simulations through poker hand evaluation. its just a small project so I can better learn MC Sims

## What it does right now
- Creates a deck of cards
- Shuffles and deals hands
- Sets up community cards
- Hand ranking logic
- Winner detection
- Monte Carlo simulation to calculate win probabilities across thousands of hands
- User Inputted Hand Evaluation
- Visualises odds using Matplotlib

## What's left to do
- Multiple Hand Evaluation
- Some Error Handling
- Add probability of each type of hand rank
- Adding More interactiveness, maybe a gui
